## 3. C 方向 MVP 范围（如果激活）

### 必须做（MVP-1 · 2-3 周）

1. **数据源**：
   - 选项 A：买第三方 API（飞瓜 / 新榜开放 API），月费 ¥500-2000，**法律 + 稳定性都最优**
   - 选项 B：自己爬抖音/小红书 trending，**有封号风险 + 反爬投入**
   - **首选 A**（用 5% ARPU 换 80% 风险消除）

2. **分析层**：每条 trending 视频 → LLM 分析输出
   - 为什么火（情绪触发：愤怒/共鸣/猎奇/实用）
   - 公式归类（挑衅断言型 / 日常洞察型 / 教学型）
   - "改造到 [用户行业]"的 3 个具体建议

3. **推送**：每日早 9:00 微信通知 / 邮件 / app 内 5 条选题

4. **复用 v2 thin slice 资产**：
   - LLM 抽象层 ✅
   - Suppression scanner ✅（用户最终生成内容时仍要扫）
   - Auth + RLS ✅
   - 删掉的：Quick Create UI（C 不需要）/ 脚本结果页（重构成"选题卡片页"）

### 不做（推迟到 MVP-2）

- 写脚本（用户自己用 Kimi / 豆包写，C 方向不抢这块）
- 视频生成（C 方向不接视频）
- 自动发布（C 方向不接发布）
- 选题趋势预测（数据量不够）

## 4. C 方向的真实问题（必须诚实回答）

| 问题 | 风险等级 | 思考 |
|---|---|---|
| 飞瓜 / 新榜 / 灰豚已经做选题了，差异化在哪？ | 🔴 致命 | 必须答得出来，否则 C 方向不成立 |
| B2B SaaS 用户真愿意付 ¥99-299 看选题吗？ | 🟡 高 | v1 访谈没问过这条，要在 V3-03 补 |
| LLM 分析成本是否真的低于视频生成？ | 🟡 中 | 1 条选题深度分析 ≈ 2000 token，按 Kimi ¥0.012/1k token = ¥0.024，¥99 ARPU 跑 4000 条都不亏 — 看起来 ✅ |
| 数据源稳定性（飞瓜涨价 / 抖音改 API） | 🟡 中 | 多源备份策略 |

### 差异化候选（待你判断）

- **a. 行业垂直化**：只服务 B2B SaaS（飞瓜/新榜服务大众）
- **b. "改造建议" 是核心**：竞品给 trending 不给"怎么改"，我们给
- **c. 推送式 vs 查询式**：竞品要用户主动查，我们每天主动推
- **d. 价格锚定低**：¥99 起 vs 飞瓜专业版 ¥598+
