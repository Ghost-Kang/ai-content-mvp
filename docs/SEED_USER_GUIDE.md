## 一、5 步产出一条视频导出包（在网页上做）

1. **打开**邀请邮件里的链接 → Clerk 一键登录（用任意邮箱，验证码即可）
2. **首页 → 「热门选题」** → 看 4 个平台 T-3 日榜，每条卡有「📊 AI 分析」按钮可看「为什么火 / 怎么改造为我自己的内容」（也可以跳过这步，直接 §3 输自定义主题）
3. 选中一条 → 点「用这条 →」 → 跳到「新建工作流」 → **可改可不改**主题文字 → 「开始」
4. 等 12 分钟左右（界面会实时显示 5 个节点状态）：选题 → 脚本 → 分镜 → 视频 → 导出
5. **导出节点完成后**，点击下载按钮拿到一个 zip 包（约 20 MB）

> ⏱️ **总时长预期**：~12-15 分钟。Seedance 视频生成是大头（17 帧 × 30-40 秒）。
> 启动后可以离开做别的事，状态不会丢。
>
> 💰 **成本透明**：每条视频默认 480p、约 ¥7-8 LLM + Seedance token 费用。
> 内测期我帮你包了，你只管用。

## 二、在剪映里完成最后 5 分钟（导入 + 配音 + 导出）

> ⚠️ **必读**：视频片段是 AI 生成、**没有声音也没有字幕**——这两步必须在剪映里手工完成。
> 操作不复杂，但每个步骤都不能省（合规字幕**强制要求**）。

### 2.1 解压 + 导入工程

1. 把 zip 完整解压到**任意非中文路径**（如 `~/Downloads/my-video/`）
2. 打开**剪映专业版**（≥ 4.0 版本）
3. 顶部菜单 **「文件」→「导入」→「Final Cut Pro XML」** → 选解压目录里的 `project.fcpxml`
4. 时间轴自动出现 17 帧视频片段，文件夹里的 `clips/frame-NN.mp4` 已被识别

> 也可以直接拖拽 `project.fcpxml` 到剪映窗口。

### 2.2 导入字幕（两次，都不能省）

剪映对 FCPXML 自带字幕兼容不好，所以字幕单独以 SRT 文件给你：

5. 时间轴空白处 → **「文件」→「导入」→「字幕」** → 选 `subtitles/disclosure.srt`
   → 全片合规字幕「本视频由 AI 辅助生成」自动铺到字幕轨。**这条不能省，不导入 = 实际不合规**（深度合成管理规定 第十七条）。
6. 重复一次：**「文件」→「导入」→「字幕」** → 选 `subtitles/narration.srt`
   → 17 条逐帧旁白字幕落到时间轴

### 2.3 加配音（三选一）

视频 mp4 里没有原生音轨。任选一种方式补上声音：

- **A · 剪映 TTS（最快）** ★ 推荐第一次试
  右键 `narration.srt` 字幕轨 → **「文本朗读」** → 选个 AI 音色 → 一键全片配音

- **B · 真人录音** （如果你想要更自然的声音）
  按字幕轨上的时间码每帧录一段，drag-and-drop 到对应时间位置

- **C · 找配音同事** （如果你有人）
  把 `script.txt` 和 `clips/` 文件夹打包发过去，时间码已对齐

### 2.4 导出最终视频

7. 顶部「导出」按钮 → 1080×1920 9:16 / 30fps（默认）→ 等渲染完成
8. 拿到一条带配音、带合规字幕、可直接发抖音 / 小红书的 mp4

## 四、常见问题

**Q: 主题/脚本不喜欢，能改吗？**
A: 可以。「新建工作流」表单里的主题就是 LLM 输入；想精改脚本，目前 MVP-1 不开放
（设计权衡：让 AI 一次性给到位）。如果脚本太离谱，回主页再开一条 run 即可。

**Q: 视频画面不喜欢能换吗？**
A: 单帧重渲是 W5 之后的功能。当前 MVP-1 走「整 run 重跑」路径——成本上你不用担心，
内测期我们包了。

**Q: 卡了 15 分钟还没出 zip，怎么办？**
A: 看「我的工作流」页面的状态：如果是 `failed`，点「重试」；如果是 `running` 但
长时间不动，截图状态发到内测群我们看一下。Seedance 偶尔会限流，我们会改去用其他
provider。

**Q: 导出的视频可以商用吗？**
A: **可以**——前提是你按 §2.2 步骤 5 导入了合规字幕。SaaS 自家产品营销是预期场景。
不要拿去做内容农场 / 二次销售，我们在「使用条款」里有写明。

**Q: 我能要更高画质吗？**
A: 1080p 在做。内测期默认 480p（成本：¥7-8/条），1080p 上线后会进入付费档（预计 ¥18-25/条）。
